# Examples

In this directory you can find some example patterns that you can use as a starting point.

## Prompts

### teasing_ride

```bash
python main.py --freq-range 500 1500 TTTGTGGTGHO 300 -o teasing_ride.mp3
```

### masochism

```bash
python main.py --freq-range 500 1500 THPPSPTXSO 300 -o masochism
.mp3
```

### rollercoaster

```bash
python main.py --freq-range 500 1500 TGHTPHTPTPHO 300 -o rollercoaster.mp3
```
